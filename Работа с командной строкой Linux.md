Ctrl + Alt + T - вызов терминала в Linux (Ubuntu)
pwd — где ты
ls — что в папке, в которой ты находишься (вывести список) (нельзя посмотреть содержимое папки на 2 шага вперед по пути, только на 1 шаг)
whoami — кто ты (вывести название учетной записи, под которой ведется работа)

mkdir - создать папку (папка = директория)
cd (папка) - перейти в эту папку (директорию)
cd (без продолжения) - перейти в самое начало пути к папке (выйти на рабочий стол)
touch - создать файл в папке (директроии) (Обязательно указать формат файла (.txt и тд)) (например touch file.txt)

cp - (copy) копирование файла в другую папку без удаления исходника (cp (файл) (папка/))
mv - (move) перемещение файла в другую папку с удалением исходника (mv (файл) (папка/))
rm - удаление файла из папки (rm (файл.формат))

ls -l - показать все права пользователя в этой папке ко всем файлам:
	d — указывает, что это каталог (директория), а не обычный файл.
	rwx — права для владельца каталога: чтение, запись и выполнение (для каталогов «выполнение» означает право входить в каталог и просматривать его содержимое). 
	r-x — права для группы пользователей: чтение и выполнение (но не запись, то есть создание или изменение файлов в каталоге). 
	r-x — права для остальных пользователей: чтение и выполнение (также без права записи).
chmod (цифра) (файл.формат) - установить права доступа для файла
chmod 777 (файл.формат) - выдать полный доступ (rwx) к файлу всем пользователям (владельцу; указанной группе пользователей; всем остальным пользователям)
chmod 644 (файл.формат) - вернуть права в изначальное состояние
 
chmod 755 (файл.формат) - стандартные права для каталогов в Linux часто задаются как drwxr-xr-x
Это означает, что владелец имеет полные права (чтение, запись и выполнение), а группа и остальные пользователи — только право на чтение и выполнение.
При таких правах владелец может создавать, удалять и переименовывать файлы внутри каталога, а другие пользователи могут просматривать содержимое каталога и получать доступ к файлам, но не могут изменять структуру самого каталога.

ss -tulnp  - посмотреть информацию о портах в linux 
top - посмотреть все процессы, которые сейчас происходят (нажать "q" для остановки)

==Для того, чтобы работать в командной строке от имени администратора необходимо перед командой добавлять sudo. Для того, чтобы зайти под правами администратора, необходимо сначала задать пароль для root (sudo passwd root)
